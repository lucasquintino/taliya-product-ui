# @taliya/tokens

Foundations do design system:

- cores;
- tipografia;
- espacamento;
- radius;
- sombras;
- bordas;
- foco;
- motion;
- densidade;
- status semanticos.

Este pacote nao depende de nenhum outro pacote do projeto.

Importe o CSS de tokens uma vez no app consumidor, antes de `@taliya/ui/styles.css` e `@taliya/crm/styles.css`:

```ts
import "@taliya/tokens/tokens.css";
```
