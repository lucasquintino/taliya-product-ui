# apps/docs

Catalogo visual do Taliya Product UI.

Storybook com exemplos de foundations, primitives, shells, padroes CRM, setup, billing, uso/cotas, agentes, configuracoes, relatorios e cobertura da matriz P0/P1/P2.

Este app pode importar `@taliya/tokens`, `@taliya/ui` e `@taliya/crm`.

```text
pnpm --filter @taliya/docs storybook
pnpm --filter @taliya/docs build
pnpm --filter @taliya/docs test:smoke
```
